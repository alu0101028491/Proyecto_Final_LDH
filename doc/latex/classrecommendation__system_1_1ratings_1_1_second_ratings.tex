\doxysubsection{Detailed Description}
Class to do many of the calculations focusing on computing averages on movie ratings. -\/ Phase 2 

The class contains the necessary methods for\+: 
\begin{DoxyItemize}
\item Get the number of films to analyze.  
\item Get the number of raters to analyze.  
\item Get the average rating given to a movie by its raters.  
\item Get the rating of movies with their average.  
\item Obtain the title of a movie by its ID and vice versa.  
\end{DoxyItemize}
