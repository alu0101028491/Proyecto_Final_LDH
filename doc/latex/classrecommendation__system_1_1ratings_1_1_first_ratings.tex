\doxysubsection{Detailed Description}
Class to answer simple questions about movies and ratings. -\/ Phase 1 

The class contains the necessary methods for\+: 
\begin{DoxyItemize}
\item Process every record from the CSV file and return an Array\+List of type Movie.  
\item Determine the maximum number of movies by any director.  
\item Load Raters.  
\item Get the number of ratings for a particular rater.  
\item Find the maximum number of ratings by any rater.  
\item Find the number of ratings a particular movie has.  
\item Determine how many movies have been rated.  
\end{DoxyItemize}
