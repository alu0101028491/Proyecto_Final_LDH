\doxysubsection{Detailed Description}
Plain Old Java Object (POJO) class for storing the data about one rating of an item An immutable passive data object (PDO) to represent the rating data 

The class contains the necessary methods for\+: 
\begin{DoxyItemize}

\item Two getter methods get\+Item and get\+Value.  
\item A to\+String method to represent rating information as a String.  
\item A compare\+To method to compare this rating with another rating.
